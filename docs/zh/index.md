# sembr 中文文档

> 基于 Reverse RAG 的意图驱动新闻监控工具。

**sembr** 让你只需写一次自然语言意图——*"监控美联储政策对新兴市场汇率的影响"*——就能持续收到语义匹配的新闻推送（Telegram / Discord / Slack），无需关键词列表，无需手动调整过滤器。
